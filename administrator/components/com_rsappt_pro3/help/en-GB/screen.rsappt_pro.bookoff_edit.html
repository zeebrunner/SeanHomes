Book-Off Edit

Description

A Book-Off is a specific date or dates that you do not want to allow bookings for a resource. Examples would be holidays, vacation days, etc.

You can Book-Off a full day or a range of hours within a day.

 

Toolbar Icons

Cancel: Abort changes and return to the previous screen.

Save: Save changes and return to the previous screen.

Help: Opens this help screen.

 

Edit Area

Resource: In edit mode this will show the name of the resource owning this book-off record. In Add New mode, you select the specific resource name.

Date: Date for the Book-Off, use the popup calendar to choose a date.

If you are entering a NEW book off, there will be 2 date boxes. You can enter a range of dates and the system will create multiple book-off days*. This would be used to book-off a week at one time. The maximum book-offs that can be made at one time is 20.

*Note: This makes multiple individual book-offs. To remove or change any or all you will need to edit/delete them individually.

 

Full Day: Yes = this book-off will block all time slots for the entire day. If Full Day = No, use the start/end times to determine the book-off start and ending times. These times are ignored if you set Full Day = Yes.
You must set the book off range to match time slot boundaries. For example if you have a time slot that runs from 10:00-11:00 do not start you book-off at 10:30.

Start/End: If 'Full Day' is No, these are the hours that will be booked off.

Description: Text entered here will appear in the book-off block on the booking screen. You can leave it blank or enter a reason for the book-off if you want the customer to know why there are no slots available.

Published: Only Book-Offs with Published set to Yes will effect the booking screens.
