Extras

Overview

Extras are OPTIONAL - you do not need to set up any extras.

Extras is used in conjunction with PayPal and the appropriate totals will be calculated.

To disable the Extras capability, do not publish any Extras.

 

 

 

Toolbar Icons

Publish: Sets one or more extras to Unpublished. Only Published extras will appear on the Appointment Booking screen.

Unpublish: Sets one or more extras to Unpublished so they will not show in the Appointment Booking screen.

Remove: Deletes one or more extras.

Edit: Opens the extra Edit screen to allow you to view and modify the extra.

New: Opens the New extra Edit screen to allow you to add a new extra.

Help: Opens this help screen.

 

List Area

ID: Unique ID for this extra.

Screen Label: This is the booking screen label for the extra.

Extra Charge: Charge per hour or per booking, for this extra.

How to Charge: per Hour or Flat rate per booking.

Order: Order extra will appear if nore than one is displayed.

Published: Indicates the published state.

Click on the extra to open the edit screen.
