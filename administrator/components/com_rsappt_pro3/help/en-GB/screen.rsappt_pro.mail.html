Appointment Booking Pro Message Centre

Description

This screen is used to setup the component messages to your particular needs.

The 'Global' message set can be edited but not deleted.

In the resource and category setup screens you can specify a spcific message set to be used, or you can use the Gloabl set.

 

Toolbar Icons

New: Creates a new message set.

New from Global: Creates a new message set using the Gloabl message text as a starting point.

Edit: Opens the message set Edit screen to allow you to view and modify the message set.

Publish: Sets one or more message sets to Unpublished. Only Published message set will appear in the dropdown list on the Resource and Category setup screens.

Unpublish: Sets one or more message set to Unpublished so they will not show in the dropdown list on the Resource and Category setup screens.

Copy Message Set: This allows you to create new message set as a copy of an existing one.

Remove: Deletes one or more message sets.

Help: Opens this help screen.
