Seat Adjustments

Description

Seat Adjustments is an optional feature allowing you to define adjustments to a resource's Max Seats setting. The adjustment can be applied based on day of the week, time of day, or day and time.

Initial Max Seats setting is done in the resource setup screen.

There are 3 ways to apply adjustments:

  1. Day Only: The adjustment will be applied to all timeslots in the selected day of the week, regardless of time.
  2. Time Only: The adjustment will be applied to all timeslots in the selected time range, every day of the week.
  3. Day and Time: The adjustment will be applied on the selected day(s) of the week, for timeslots in the specified time range.

Note: Timeslots that begin or end in an adjustment area will have the adjustment applied.
For example, if an adjustment is set as 4:00-6:00, a slot that runs from 3:00 to 4:30 will have the adjustment applied.


Toolbar Icons

Publish: Sets one or more rate adjustments to Published.
Unpublish: Sets one or more rate adjustments to Unpublished.
Remove: Deletes one or more rate adjustments.
Edit: Opens the rate adjustments Edit screen to allow you to view and modify the adjustment.
New: Opens the New rate adjustments screen to allow you to add a new adjustment.
Help: Opens this help screen.

 

List Area

ID: Unique ID for the adjustment rate.
Resource: This is the resource that is getting the adjustment rate.
By: This is when the adjustment will be applied.
Days: Shows which days of the week to apply the adjustment for Day Only or Day and Time.
Time Range: Shows the time range to be used for Time Only or Day and Time.
Adjustment: Shows the adjustment amount. Negative adjustments are supported and allow a reducution of Max Seats.
Published: Indicates the published state.

Click on the ID to open the edit screen.
