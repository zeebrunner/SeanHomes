Seat Adjustments

Description

Seat Adjustments is an optional feature allowing you to define adjustments to a resource's Max Seats setting. The adjustment can be applied based on day of the week, time of day, or day and time.

Initial Max Seats setting is done in the resource setup screen.

There are 3 ways to apply adjustments:

  1. Day Only: The adjustment will be applied to all timeslots in the selected day of the week, regardless of time.
  2. Time Only: The adjustment will be applied to all timeslots in the selected time range, every day of the week.
  3. Day and Time: The adjustment will be applied on the selected day(s) of the week, for timeslots in the specified time range.

Note: Timeslots that begin or end in an adjustment area will have the adjustment applied.
For example, if an adjustment is set as 4:00-6:00, a slot that runs from 3:00 to 4:30 will have the adjustment applied.

Negative adjustments are supported and allow a reducution of Max Seats.
