PayPal Transactions

Description

All PayPal transaction data is stored for your reference. This information is read-only and is created by PayPal.

 

Toolbar Icons

Remove: Deletes one or more transaction records.

View: Opens the PayPal Transaction View screen to allow you to view details for the transaction.

Export/CSV: Exports transaction details for one or more PayPal Transactions to a csv file. The csv file can be read into Excel for reporting purposes.

Help: Opens this help screen.

 

List Area

PayPal txnID: Unique ID, assigned by PayPal, for the transaction.
Request ID: The Booking Request to which the PayPal transaction refers.
Buyer's Name: Name of the buyer in the transaction.
Email: Email address of the buyer in the transaction.
Date: Transaction date.
Status: Transaction status.
Total, Fee & Tax: Amounts charged to the customer. Tax settings are controlled in you PayPal account.

Click on the PayPal txnID link to open the view screen.
Click in the Request ID link to open the request detail screen.
